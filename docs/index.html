VAIS — Voynich Analysis Interactive Suite

A reproducible laboratory for statistical analysis of the Voynich manuscript (Beinecke MS 408): 147 runnable tests with pre-registered kill criteria, calibrated control corpora, golden reference outputs, and an adjudicated research program. Nothing here claims a decode; the methodology is the point.

Every test page has a ▶ Run button that executes the real instrument in your browser (Python on WebAssembly — nothing uploaded, nothing installed), plus copy-paste instructions for running it locally at full speed.
