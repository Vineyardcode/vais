ROOT-TYPE GRAMMAR — the substance/process root split
========================================
Dig deeper into the Type A (substance) / Type B (process) root split.

Tests:
  1. TYPE A/B VALIDATION — Is the split bimodal or a continuum?
  2. LINE-LEVEL GRAMMAR — Do A and B roots alternate in lines? (noun-verb-noun)
  3. THE q- PREFIX ANOMALY — Why does q- overwhelmingly select suffix -l?
  4. SUFFIX PARADIGM STRUCTURE — Are -iin, -r, -l, -in, -m truly "cases"?
  5. LINE-POSITION EFFECTS — Does -y appear more in certain positions?
  6. PROCESS-ROOT SEQUENCES — What happens around ched/kee/keed clusters?

Parameters

Edit a value to override it for an in-browser run (the script itself is never modified).

nametypevalue
PREFIXESjson
ROOT_ONSETSjson
ROOT_BODIESjson
SUFFIXESjson

Run it in your browser

Python + numpy on WebAssembly, entirely on your machine — nothing is uploaded. First run downloads the runtime (~15 MB) and the repository data pack (~9 MB); both are cached afterwards. Expect it to be slower than a native run.




Run it locally (full speed, byte-exact)


git clone https://github.com/Vineyardcode/vais
cd vais
pip install numpy flask
python scripts/root_type_grammar.py        # this test; output also lands in results/
python webui/server.py          # or the full interactive UI at localhost:5000
