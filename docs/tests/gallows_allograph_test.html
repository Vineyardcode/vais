QUESTION: gallows_line_position_split proved {p,f} are first-line characters and {t,k}
are body-text characters. But WHY? Two competing hypotheses:

  H1 (ALLOGRAPH): p is just "first-line t" and f is "first-line k" ‚Äî
      the same underlying letter written differently on line 1.
      Prediction: replacing p‚Üít in L1 words produces valid L2+ words.
      Context distributions (successor/predecessor) should match.

  H2 (INDEPENDENT): p and f are separate symbols from t and k, with
      different functions (p=process, f=botanical vs
      t=celestial, k=generic).
      Prediction: p‚Üít substitution produces nonsense; contexts diverge.

  H3 (PREFIX/MARKER): p and f are structural markers ADDED to words
      on L1 ‚Äî not substituting for anything.
      Prediction: STRIPPING p/f from L1 words yields valid words.

TESTS:
A. SUBSTITUTION TEST ‚Äî Replace p‚Üít in L1 words. What fraction match
   real L2+ vocabulary? Controls: p‚Üík, f‚Üík, f‚Üít, random‚Üírandom.

B. CONTEXT DISTRIBUTION ‚Äî JSD between successor distributions:
   {glyph after p on L1} vs {glyph after t on L2+},
   {glyph after f on L1} vs {glyph after k on L2+}.
   If allographs, JSD should be low (similar contexts).

C. DELETION TEST ‚Äî Strip p/f glyphs from L1 words. What fraction of
   residues are valid L2+ words? Compare with stripping t/k.

D. SLOT POSITION TEST ‚Äî In the IMF grammar, do p and t occupy the
   same positional slots within words? Do f and k?

E. SEMANTIC RETEST ‚Äî Redo gallows_semantics semantic domain analysis using ONLY
   L2+ data. If p still shows "process" enrichment without L1, the
   semantic classification is real, not a positional artifact.

F. WORD-PAIR MATCHING ‚Äî For each L1 word containing p, find the most
   similar L2+ word (edit distance). Is that word a t-for-p variant?
