DISCRIMINATING HYPOTHESES
======================================

Post-synthesis_stress_test, we know:
  - Structure is character-level (z=80.3), NOT morphological (95.6% char-explained)
  - Vocabulary is anti-compositional (z=−25.4) under morphological decomposition
  - Can't distinguish natural from constructed language
  - Lines are syntactic units, left-to-right, gap-1 only

hypothesis_discriminators asks: what KIND of system produces these properties?
Five discriminating tests between natural language, cipher, and generation:

  A) CHARACTER MODEL vs. LEXICON
       Can a character bigram model predict which words exist?
       If vocab is generated by character rules → clean separation.
       If vocab is a codebook → no clean separation.

  B) WORD BURSTINESS
       Natural language words cluster (use once → use again soon).
       Codebook/cipher tokens are more uniformly distributed.
       Burstiness B>0 = natural, B≈0 = codebook, B<0 = mechanical.

  C) WITHIN-LINE SURPRISAL CURVE
       Natural sentences: specific surprisal profile (high start, dip, rise).
       Encoding: flat or monotonic.

  D) CHARACTER POSITIONAL ENTROPY
       H(char at position k within word) for k=1,2,...
       Natural language has specific patterns (consonant/vowel slots).
       Cipher/generated may be more uniform.

  E) CHARACTER-LEVEL COMPOSITIONALITY (redo of a synthesis_stress_test sub-analysis)
       synthesis_stress_test (part d) used morphological decomposition (now known to be mostly
       character-level). Redo purely at character level: first-2 × last-2
       combinations. If vocabulary is compositionally generated at char
       level: high fill. If closed lexicon: low fill.

Key questions hypothesis_discriminators addressed:
1. Is the vocabulary generated by character-level rules, or is it a fixed list?
2. Do words cluster in usage (natural language) or distribute uniformly (codebook)?
3. Do lines have sentence-like information structure?
4. Are within-word character positions phonotactically constrained?
5. Is the vocabulary compositional at the character level (not just morphological)?

Implications for hypothesis discrimination:
- Natural language: expects burstiness, declining surprisal, phonotactic positional H
- Cipher/encoding: expects uniform burstiness, flat surprisal, uniform positional H  
- Mechanical generation: expects regularity, possibly cyclic patterns
