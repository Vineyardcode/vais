zodiac_label_network

scripts/zodiac_label_network.py · depends on ring_decan_mapping

Cross-Sign Label Network Analysis

Questions:
  1. Which zodiac signs share the most labels? Does sharing follow
     astrological geometry (triplicities, oppositions, sextiles)?
  2. Do shared labels cluster at specific degree positions?
  3. What is the "vocabulary fingerprint" of each sign?
  4. Can we identify label "families" that span the zodiac in regular patterns?
  5. Do degree-position pairs (e.g., "degree 10 in any sign") share roots?

This builds on all prior analyses: label extraction (pipeline), cross-reference
(astro_crossref), ring mapping (ring_decan), and medieval degrees.

Run it in your browser

Python + numpy on WebAssembly, entirely on your machine — nothing is uploaded. First run downloads the runtime (~15 MB) and the repository data pack (~9 MB); both are cached afterwards. Expect it to be slower than a native run.




Run it locally (full speed, byte-exact)


git clone https://github.com/Vineyardcode/vais
cd vais
pip install numpy flask
python scripts/zodiac_label_network.py        # this test; output also lands in results/
python webui/server.py          # or the full interactive UI at localhost:5000
