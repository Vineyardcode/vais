morphology_full_survey

scripts/morphology_full_survey.py

Full Morphological Attack Plan

Executes the 5-step decipherment strategy:
  1. Improved parser (fixes sh-words, al/ol/ar/or endings)
  2. Full paradigm/declension tables per root
  3. Root–imagery correlation (labels on herbal/pharma pages)
  4. Prefix substitution hypothesis test (qo- vs o- etc.)
  5. I-series mapping (daiin vs dain vs daiiin)

Based on Zattera glyph-slot system.

Parameters

Edit a value to override it for an in-browser run (the script itself is never modified).

nametypevalue
PREFIXESjson
ROOT_ONSETSjson
ROOT_BODIESjson
SUFFIXESjson

Run it in your browser

Python + numpy on WebAssembly, entirely on your machine — nothing is uploaded. First run downloads the runtime (~15 MB) and the repository data pack (~9 MB); both are cached afterwards. Expect it to be slower than a native run.




Run it locally (full speed, byte-exact)


git clone https://github.com/Vineyardcode/vais
cd vais
pip install numpy flask
python scripts/morphology_full_survey.py        # this test; output also lands in results/
python webui/server.py          # or the full interactive UI at localhost:5000
