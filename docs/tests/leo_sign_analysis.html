leo_sign_analysis

scripts/leo_sign_analysis.py

Leo Zodiac Folio Deep-Dive

Focused decipherment attempt on the Leo zodiac folio.

Tests:
  1. FULL FOLIO DECOMPOSITION — every Leo label parsed into root+suffix
  2. VOCABULARY MATCHING — Leo roots vs the expanded root lexicon
  3. ch=kh HYPOTHESIS — does the 'hot' reading fit a fire sign?
  4. ROOT DISTRIBUTION — where else do Leo roots appear in the manuscript?
  5. LEO-EXCLUSIVE ROOTS — vocabulary unique to this folio within the zodiac

Parameters

Edit a value to override it for an in-browser run (the script itself is never modified).

nametypevalue
SIMPLE_GALLOWSjson
BENCH_GALLOWSjson
COMPOUND_GCHjson
COMPOUND_GSHjson
PREFIXESjson
SUFFIXESjson

Run it in your browser

Python + numpy on WebAssembly, entirely on your machine — nothing is uploaded. First run downloads the runtime (~15 MB) and the repository data pack (~9 MB); both are cached afterwards. Expect it to be slower than a native run.




Run it locally (full speed, byte-exact)


git clone https://github.com/Vineyardcode/vais
cd vais
pip install numpy flask
python scripts/leo_sign_analysis.py        # this test; output also lands in results/
python webui/server.py          # or the full interactive UI at localhost:5000
